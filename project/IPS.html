What is KaboosIPS

KaboosIPS is the foremost Open Source Intrusion Prevention System (IPS) in the world. KaboosIPS IPS uses a series of rules that help define malicious network activity and uses those rules to find packets that match against them and generates alerts for users.


KaboosIPS can be deployed inline to stop these packets, as well. KaboosIPS can be downloaded and configured for personal and business use alike.
